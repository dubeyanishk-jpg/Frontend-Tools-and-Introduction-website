HTML

The foundation of the World Wide Web

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It organizes content—like text, links, and images—for display in web browsers. HTML works alongside CSS for styling and JavaScript for functionality.

Core Concepts

  • Tags & Elements: Represented by opening and closing tags.
  • Structure: Defines headings, paragraphs, links, etc.
  • Attributes: Provide extra information (e.g., src, href).

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
   <head>
      <title>Page Title</title>
   </head>
   <body>
      <h1>Heading</h1>
      <p>Paragraph</p>
   </body>
</html>

Key Features

  • <h1> to <h6> → Headings
  • <p> → Paragraphs
  • <a href=""> → Links
  • <img src="" alt=""> → Images
  • <ul>, <ol>, <li> → Lists

HTML5 & Modern Web Standards

HTML5 introduced semantic elements like <article>, <section>, <nav>, and <footer>. It also supports native audio/video embedding and APIs like geolocation and local storage.

Accessibility & ARIA

Developers use WAI-ARIA to make websites accessible. Attributes like aria-label and aria-expanded provide context for assistive technologies.
